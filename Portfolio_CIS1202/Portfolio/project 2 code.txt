  <section id="intro">
    <h2>Introduction</h2>
    <p>
       This <strong>School Computer Lab Network Manual</strong> is designed to help students and staff understand how the school’s computer lab network operates. 
      It covers the basic network types, topologies, technologies, and tools used to maintain smooth and secure connections. 
      The manual also guides students on proper handling of cables and performing simple network checks before using the lab equipment.
    </p>
  </section>

  <section id="types">
    <h2>Types of Networks</h2>
    <p>Our school network is made up of several interconnected systems. Below are the network types used within our school setup:</p>
    <dl>
      <dt>LAN (Local Area Network)</dt>
      <dd>Connects all computers, printers, and the server within the computer lab or building.</dd>

      <dt>WAN (Wide Area Network)</dt>
      <dd>Connects the school’s local network to the Division or DepEd network for internet and resource sharing.</dd>

      <dt>MAN (Metropolitan Area Network)</dt>
      <dd>Used when connecting nearby schools or educational offices within the same city.</dd>

      <dt>PAN (Personal Area Network)</dt>
      <dd>Used for short-distance communication between personal devices such as laptops or phones.</dd>
    </dl>
  </section>

  <section id="topologies">
    <h2>Network Topologies</h2>
    <p>
      The school’s computer lab follows a <strong>Star Topology</strong> layout. All computers are connected to a central hub or switch that distributes data across the network.
      This structure makes it easier to detect and fix issues—if one computer or cable fails, the rest of the network stays connected.
    </p>

    <figure>
      <img src="star topology.png" alt="Diagram of Star Topology in the School Lab">
      <figcaption>Figure 1: Star Topology – Common network layout used in the computer lab.</figcaption>
    </figure>
  </section>

  <section id="technologies">
    <h2>Networking Technologies</h2>
    <p>
      The lab uses <strong>Ethernet (Cat6)</strong> for wired connections and <strong>Wi-Fi</strong> for wireless access. 
      Each workstation is connected through a switch to ensure reliable communication. 
      A firewall router controls data flow, providing both internet access and protection.
    </p>
    <p>
      The school’s server stores student files, and the system uses IP addressing and DHCP to automatically assign connections to devices.
    </p>

    <figure>
      <img src="0a7df387-0a48-455e-ad47-97402b0dd12c.png" alt="Cat6 Ethernet Cable">
      <figcaption>Figure 2: Cat6 Ethernet Cable — used for wired connections in the computer lab.</figcaption>
    </figure>
  </section>

  <section id="tools">
    <h2>Network Tools and Equipment</h2>
    <p>These are the key tools used to maintain and repair the computer lab network:</p>
    <ul>
      <li><strong>Crimping Tool:</strong> Attaches RJ45 connectors to Ethernet cables.</li>
      <li><strong>LAN Tester:</strong> Checks if cables are properly wired and working.</li>
      <li><strong>Wire Stripper:</strong> Removes cable insulation for connector attachment.</li>
      <li><strong>Patch Panel:</strong> Keeps cable connections neat and organized.</li>
      <li><strong>Switch and Router:</strong> Manage data transmission and internet access.</li>
    </ul>
  </section>
